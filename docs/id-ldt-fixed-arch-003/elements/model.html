EDIC Local Digital Twin - Reference Architecture
The EDIC Local Digital Twin - Reference Architecture describes the architectural choices for the LDT Citiverse EDIC. It consists of: - A metamodel, to focus the architectural patterns being used for the reference architecture. - Level 0, motivational and strategy layers viewpoint. - Level 1, Strategy, Business and Application layers viewpoint. At this moment (Q1/Q2 2026) this architecture is very much a draft and it is actively being worked on.
