Calculation ()
> Applying calculation rules and models to data to generate results that provide insight into processes and effects. The calculation function uses models to provide insight into phenomena like heat stress or waterlogging. It converts raw data into indicators, which are essential for policy-based assessments. These models are transparent and documented so that the underlying assumptions are clear.
