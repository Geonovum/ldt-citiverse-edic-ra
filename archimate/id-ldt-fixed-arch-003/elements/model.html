EDIC Local Digital Twin - Reference Architecture
The EDIC Local Digital Twin - Reference Architecture describes the architectural choices for the LDT CitiVERSE EDIC. It consists of: - A metamodel, to focus the architectural patterns being used for the reference architecture. - Overall architecture views. - Architecture views for the EDIC organisation and infrastructure itself -> in the Views / EDIC folder - Architecture views for the LDT reference patterns and buildingblocks -> in the Views / LDT-reference architecture folder - Architecture views for solution architectures and how they relate to the reference architecture -> in the Views / LDT-solutions folder This reference architecture is maintained in Archi using the coArchi Collaboration plugin in the https://github.com/Geonovum/ldt-citiverse-edic-ra repository in the master branch. The reference architecture is published using the Archi HTML report engine using github.io from the main branch in the same repository. Comments and feedback can be provided using the issues functionality in the same repository: https://github.com/Geonovum/ldt-citiverse-edic-ra/issues
