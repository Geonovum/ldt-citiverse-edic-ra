# ldt-citiverse-edic-ra

__NOTE: Temporary repository for LDT Citiverse EDIC Reference Architecture__

A more permanent repository will be established within the EDIC Infrastructure within due time.

The reference architecture can be viewed here: https://geonovum.github.io/ldt-citiverse-edic-ra/


The Reference Architecture currently has 3 views:
- [Metamodel](https://geonovum.github.io/ldt-citiverse-edic-ra/?view=id-75cbd05360cb40f6b4e27d45af3cb46b) this viewpoint expresses the elements and relations that we use in this reference architecture
- [level 0](https://geonovum.github.io/ldt-citiverse-edic-ra/?view=id-view-ldt-full) this viewpoint shows the Motivation and Strategy layers
- [level 1](https://geonovum.github.io/ldt-citiverse-edic-ra/?view=id-b11dfbe262554939b4af21baa957b2e7) this viewpoint shows the Strategy, business and application layers

These views are a first draft for the reference architecture, by no means complete yet.
